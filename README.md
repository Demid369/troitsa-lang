# Троица

**Русскоязычный язык программирования с троичной логикой**

```троица
печать("Привет, мир!")

пусть ответ = Может  // Да, Нет или Может

если ответ == Да {
    печать("Утверждение")
} иначе если ответ == Нет {
    печать("Отрицание")
} иначе {
    печать("Неопределённость")
}
```

## Особенности

- **Русский синтаксис** — код на родном языке
- **Троичная логика** — три состояния: `Да`, `Нет`, `Может`
- **Классы и объекты** — ООП с русскими ключевыми словами
- **Словари и списки** — встроенные структуры данных
- **Модули** — импорт и экспорт функций

## Синтаксис

### Переменные

```троица
пусть имя = "Троица"
пусть число = 369
пусть трит = Да
```

### Троичная логика

```троица
пусть а = Да
пусть б = Может

печать(а и б)        // Может (MIN)
печать(а или б)      // Да (MAX)
печать(не а)         // Нет
печать(консенсус(а, б))  // Может
```

### Функции

```троица
функция факториал(н) {
    если н <= 1 { вернуть 1 }
    вернуть н * факториал(н - 1)
}

печать(факториал(5))  // 120
```

### Классы

```троица
класс Точка {
    пусть х = 0
    пусть у = 0
    
    функция создать(х, у) {
        этот.х = х
        этот.у = у
    }
    
    функция расстояние() {
        вернуть корень(этот.х * этот.х + этот.у * этот.у)
    }
}

пусть т = новый Точка()
т.создать(3, 4)
печать(т.расстояние())  // 5
```

### Словари

```троица
пусть человек = {
    "имя": "Иван",
    "возраст": 30,
    "статус": Да
}

печать(человек["имя"])  // Иван
```

### Модули

```троица
// математика.t9
экспорт функция квадрат(х) {
    вернуть х * х
}

// главная.t9
импорт {квадрат} из "математика.t9"
печать(квадрат(5))  // 25
```

## Троичная логика

В отличие от двоичной логики (истина/ложь), троичная логика имеет три состояния:

| Значение | Число | Смысл |
|----------|-------|-------|
| `Да` | +1 | Утверждение, истина |
| `Может` | 0 | Неопределённость, баланс |
| `Нет` | -1 | Отрицание, ложь |

### Таблица истинности И (MIN)

```
      | Нет  | Может | Да
------+------+-------+-----
Нет   | Нет  | Нет   | Нет
Может | Нет  | Может | Может
Да    | Нет  | Может | Да
```

### Таблица истинности ИЛИ (MAX)

```
      | Нет   | Может | Да
------+-------+-------+----
Нет   | Нет   | Может | Да
Может | Может | Может | Да
Да    | Да    | Да    | Да
```

## Примеры

### Голосование с неопределённостью

```троица
функция голосование(голоса) {
    пусть да_счёт = 0
    пусть нет_счёт = 0
    
    для голос в голоса {
        если голос == Да { да_счёт = да_счёт + 1 }
        если голос == Нет { нет_счёт = нет_счёт + 1 }
    }
    
    если да_счёт > нет_счёт { вернуть Да }
    если нет_счёт > да_счёт { вернуть Нет }
    вернуть Может
}

пусть голоса = [Да, Да, Нет, Может, Да]
печать(голосование(голоса))  // Да
```

### Медицинская диагностика

```троица
функция диагноз(симптомы) {
    пусть уверенность = Может
    
    для симптом в симптомы {
        уверенность = уверенность и симптом
    }
    
    выбор уверенность {
        Да: печать("Диагноз подтверждён")
        Нет: печать("Диагноз исключён")
        Может: печать("Требуется дополнительное обследование")
    }
    
    вернуть уверенность
}
```

## Философия

> *"3, 6, 9 — ключ к вселенной"* — Никола Тесла

Троица основана на идее, что мир не бинарен. Между "да" и "нет" существует "может быть" — состояние неопределённости, баланса, потенциала.
